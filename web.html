COPPA.ageMeetsCOPPARequirements

function COPPA.ageMeetsCOPPARequirements(java.util.Calendar birthdate)
Is a user with the provided birthday old enough to meet COPPA requirements?
Parameters:
birthdate - the user's birth date as anything that can be parsed into a Date object via the Date.parse() method.
Returns:
